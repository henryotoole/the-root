This directory is a place to store ALL projects for viewing in the project nav.

Projects are HTML files which act much like normal HTML files except for the following:
A) To be recognized as a project, each file must have a <meta> tag with the following information:
	<meta name="proj_id" content="XXXXXXX">
	<meta name="proj_name" content="XXXXXXX">
	<meta name="cat_id" content="XXXXXXX">
	<meta name="cat_name" content="XXXXXXX">
	
	Additionally, the following optional tags are allowed
	<meta name="cat_parent" content="XXXXXXX">
